Buff寄与率・1枠効率

使用中の各行を1つずつOFFにしたときのダメージ低下量を表示します。 タグが付いている項目は、同じタグの装備/BuffをまとめてOFFにした「タグ別寄与率」として表示します。 タグが付いていない項目も、タグ別寄与率の表に単体項目として表示します。 「枠効率」は、枠を消費する行だけ「低下ダメージ ÷ 枠数」で表示します。

価値表示

現在の構成を基準に、攻撃力+1 / 魔力+1 / 与ダメ+1% などを足した時の増加量を表示します。 ステータスやBuffを変更すると、この表も自動で再計算されます。 なおAC0で倍率が同じ場合、攻撃力+1の価値は一定に見えやすいです。

武器・防具・装飾補正

右手・左手・弾丸、防具7部位、装飾7部位を固定行で用意しています。 装備欄は常に使用扱いです。未装備や補正なしの部位は空欄/0のままでOKです。 攻撃力・魔力・速度は計算に反映し、攻撃ディレイは合計値を表示します。

複合Buff：1枠で複数効果

1行 = 1Buff枠 として扱うメイン入力欄です。 1つのBuffに複数効果がある場合、ここにまとめて入力してください。 計算時は内部で割合Buff・実数加算・変換・与ダメ・特攻へ展開します。%欄は10%なら10と入力します。

使用枠名称タグ 攻撃力%魔力%速度% 攻撃力+魔力+速度+ 魔力→攻撃力%速度→攻撃力% 与ダメ%特攻倍率 メモ操作

タグ連携

装備欄と複合Buff欄で同じタグを入れると、ここにまとめて表示します。タグ名にスペースを含められます。複数タグはカンマ区切りです。 例: パンチング 魔力装備 火力A

旧項目 単体Buffも複合Buffセクションでまとめて入力してOKです

割合ステータスBuff：攻撃力% / 魔力% / 速度%

上から順番に適用。%欄は「10%なら 10」。 攻撃力%は、基礎攻撃力側から増分を出して攻撃力加算上限に含めます。

実数ステータス加算：魔力 / 攻撃力 / 速度

魔力+X、攻撃力+X、速度+Xを1つの表で管理します。 対象をプルダウンで選択してください。 魔力/速度は割合ステータスBuffやステータス変換の前に加算、攻撃力は攻撃力実数加算加算として攻撃力加算上限の対象にします。

使用枠名称対象加算値メモ / ソース操作

ステータス変換：魔力/速度/酩酊度など → 攻撃力

例: 魔力30%を攻撃力に足すなら「参照=魔力、変換率=30」。 魔力/速度/酩酊度などからの攻撃力変換は、攻撃力加算上限500を突破する上限外加算として扱います。

使用枠名称参照変換率%参照補正メモ / ソース操作

参照補正は「参照ステータスに足してから%変換」します。例: 魔力125、率10%、参照補正10 → (125+10)×10%=13.5。

種族特攻・特攻倍率：乗算倍率

1.5倍なら 1.5、2倍なら 2.0。複数行ONの場合は乗算します。 特攻はダメージ本体だけでなく、ACがある場合の防御計算側にも入ります。

使用枠名称倍率メモ / ソース削除

物理与ダメージ増加：加算倍率

+100%なら 1.00、+20%なら 0.20。複数は 1 + 合計% で加算倍率化。
