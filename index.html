最適化計算 登録済み候補からおすすめ構成を探す

登録済みの装備候補とBuff候補から、目的に合う構成を探します。 候補を段階的に絞りながら探すため、0から一発で終わらせるより、良い結果を適用してから再検索すると詰めやすいです。

最適化計算の使い方

1. 候補を用意する

装備は「装備登録」タブ、Buffは「Buff登録」タブに登録します。 候補の追加は、先に「装備登録」「Buff登録」で行ってください。ここでは登録済み候補の組み合わせを探します。

2. 現在構成を決める

各行の「使用」は現在ONの構成です。 手動で強そうな構成をONにしておくと、その構成を基準にして比較できます。 0から探す場合は、装備登録/Buff登録の「使用を全解除」ボタンで一度空にすると見通しがよくなります。

3. 固定・除外を使う

絶対に使いたい装備は、装備登録タブで「固定」をONにします。 探索に入れたくない装備は「除外」をONにします。 「使用」と「固定」は別物です。使用は「今の構成」、固定は「検索するときに必ず使う指定」です。

4. 目的を選んで検索する

目的で「最大ダメージ」「最大攻撃力」「最大クリ率」などを選びます。 目標値を入れると、単に最大値を狙うのではなく、目標に届いたうえで近い構成を優先します。

おすすめの探し方
  1. 0から探す: 装備とBuffの使用を全解除してから検索します。
  2. 良さそうな結果が出たら: 結果の「適用」を押して現在構成にします。
  3. もう一度検索: 適用した構成を基準にして再検索します。「現在の装備でBuffだけ組み直す」がONなら、その装備のままBuffだけ詰め直します。
  4. さらに詰める: 必ず使いたい装備を「固定」、使わない候補を「除外」にして、検索精度を「精度重視」にします。

体感として、0から一発で終わらせるより、検索 → 良い結果を適用 → もう一度検索 を数回回すほうが良い結果に近づきやすいです。 一度見つけた良い構成を基準にすると、さらに良い組み合わせを見つけやすくなります。

結果の見方
  • 自動探索: 最適化が自動で見つけた候補です。
  • 現在ON: 手動でONにしている現在構成です。比較用の基準です。
  • 現在ON装備 + 最適Buff: 装備は現在ONのまま、Buffだけ詰め直した候補です。
  • 現在ONより良い結果だけ表示: 現在構成を超えた候補だけ見たい時に使います。基準だけ表示される場合は、その条件では改善候補が見つかっていません。
検索がうまくいかない時
  • 結果が出ない場合は、バフ枠上限、ステータス上限、目標値、競合グループを確認してください。
  • 候補が少なすぎる場合は、装備やBuffの「除外」がONになっていないか確認してください。
  • 強い手動構成がある場合は、それを「使用」ONにしてから検索すると比較しやすくなります。
  • 探索が弱そうな場合は、検索精度を「精度重視」、Buff探索モードを「深掘り」にすると重い代わりに拾いやすくなります。

迷ったら、最初は初期値のままでOKです。結果が弱そうな時だけ「検索精度」を上げたり、「Buffの探し方」を深掘りにしてください。
