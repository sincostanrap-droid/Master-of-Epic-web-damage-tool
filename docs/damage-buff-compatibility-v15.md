# v15: ダメージ系Buff併用表の取り込み

MoE Wiki `アイテム/追加効果/併用2` の表を元に、装備Buffへダメージ関連効果を自動投入する。

## 元データ

- クリティカル率
- 効果範囲
- 物理攻撃ダメージアップ（物理与ダメージ増加バフ）
- 魔法攻撃ダメージアップ（魔法与ダメージ増加バフ）
- 火/水/風/地/無属性効果アップ
- 攻撃力変換

## 自動投入方針

物理ダメージ計算に関わる既存欄へ入れるもの:

- 物理与ダメージ増加 → `equipBuffDmgPct`
- 攻撃力変換 魔力xN% → `equipBuffConvMagicRate`
- 攻撃力変換 移動速度xN% → `equipBuffConvSpeedRate`
- クリティカル率 → `equipBuffExtraCritRatePct`

表示用に留めるもの:

- 魔法与ダメージ増加
- 属性ダメージ増加
- 効果範囲

現行ツールは物理ダメージ計算が主目的なので、魔法/属性/範囲は物理計算へ混ぜない。

## 競合グループ

Wiki表のグループを元に、確定値・確定グループ・注意備考なしのものだけ自動候補にする。

- 物理与ダメ: `damage:physical:<group>`
- クリティカル: `critical:<group>`
- 攻撃力変換: `conversion:attack:<group>`
- 魔法/属性/範囲: 表示・監視用タグ

v14でスキル+競合が先に入っている場合でも、v15では物理ダメージ計算に直接影響する `damage:physical:*` を代表競合として優先する。その他の競合軸はタグへ残す。

## レビュー出力

`dist/damage-buff-compatibility/` に確認用TSV/summaryを出力する。リポジトリに残すかは任意。
