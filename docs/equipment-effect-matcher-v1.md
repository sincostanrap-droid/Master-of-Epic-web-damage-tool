# 装備効果 突合ビュー v1

`equipment-effect-matcher.html` は、既存の装備登録データや貼り付けた装備JSON/TSVを、参考効果データ・装備効果索引と突き合わせるための確認画面です。

## 目的

- 既存の物理ダメージ計算にはまだ接続しない
- 装備名、Buff名、BuffStats、説明文から参考データ側の候補を表示する
- 装備元データから読める効果候補と、参考データ側の候補を分けて並べて確認する
- 説明文や自動計算未対応の効果を捨てずに見える化する

## 読み込み元

- ブラウザ保存データを自動探索
- 貼り付けJSON
- 貼り付けTSV
- 装備名/Buff名だけのリスト

ブラウザ保存データは、同じオリジンの `localStorage` を走査します。通常は `http://localhost:8080/` で本体を触ったあと、同じサーバー上で `http://localhost:8080/equipment-effect-matcher.html` を開くと候補を拾えます。

## 突合の考え方

スコアは以下の要素で付けています。

- 装備名の完全一致・部分一致
- Buff名の完全一致・部分一致
- BuffStats/説明文の類似
- 装備元データの効果候補と参考データ側の効果statの一致

このスコアは確定判定ではなく、目視確認のための並び替えです。

## 次の段階

この画面で候補の突合が妥当そうなら、次は既存の装備詳細UIに「参考候補を見る」ボタンを追加します。その後、攻撃力・命中・攻撃ディレイ・クリティカルだけを試験的に計算へ接続します。

## 表示上の注意

- 「装備元データから読めた効果候補」は、選択した装備raw自身のフィールド、`BuffStats`、`add_status`、`stats` などから抽出します。参考データ側の候補値を混ぜません。
- 「参考データ側の突合候補」は名前・Buff名・説明文・効果statの類似から並べ替えた候補です。確定値ではありません。
- 「説明/未対応として保持した情報」は、効果説明として存在するが、現時点では計算式や条件を決めていない情報です。
- この画面は確認専用です。既存のダメージ計算結果は変更しません。
