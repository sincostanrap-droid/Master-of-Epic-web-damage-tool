# v16: 装備カタログのスマホ向けレイアウト調整

装備カタログタブがスマホなどの狭い画面で横にはみ出す問題へのCSSのみの修正です。

## 変更内容

- カタログ検索ツールバーを画面幅に応じて折り返す
- 検索欄・セレクト・再読み込みボタンを親幅に収める
- ページ送りを狭幅画面で折り返す
- 結果テーブルは画面全体を押し広げず、テーブル部分だけ横スクロールにする
- 計算処理、カタログデータ、JSには触れない

## 確認ポイント

- スマホ幅でカタログタブを開いてもページ全体が横にはみ出さない
- 検索欄、カテゴリ、部位、効果、Buff有無、ソート、順序、1ページ、再読み込みが画面内に収まる
- 結果テーブルは横スクロールで確認できる
- 装備登録へ追加ボタンが押せる
- 他タブの表示や計算結果が変わらない
